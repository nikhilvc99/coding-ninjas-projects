# Alarm Clock 

A simple web-based alarm clock application.

## Features

- Set multiple alarms with customizable times.
- Choose between AM and PM for each alarm.
- Sound alarm with customizable audio file.
- Snooze and dismiss options for alarm alerts.

## Technologies Used

- HTML
- CSS
- JavaScript

## Usage

1. Clone the repository:
2. Open index.html in your web browser.
3. Set alarms using the provided inputs:
    • Enter the hour, minute, and second for the alarm.
    • Select AM or PM.
    • Click "Set Alarm" to add the alarm.



4. Once an alarm is set, it will trigger at the specified time.
5. To snooze or dismiss an alarm:
    • When the alarm rings, click the "Snooze" button to postpone it for a few minutes.
		• Click the "Close" button to dismiss the alarm.
